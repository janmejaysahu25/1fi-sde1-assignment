# 1Fi SDE1 Assignment — Fullstack Runnable Project

This is a runnable fullstack project tailored for the 1Fi SDE1 assignment.

## Contents
- frontend/ — React + Vite + Tailwind frontend
- backend/  — Express + MongoDB (Mongoose) backend with seed script

## Quick start (local)
1. Setup backend
   - cd backend
   - create .env with MONGODB_URI and PORT (example in .env.example)
   - npm install
   - npm run seed
   - npm run dev

2. Setup frontend
   - cd frontend
   - npm install
   - npm run dev
   - Open http://localhost:5173
